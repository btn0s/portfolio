# Echelon

Inspired by Splinter Cell's multiplayer, Echelon is a game concept I designed and prototyped in Unreal Engine Blueprints. The game pits agile spies (third-person) against well-armed mercenaries (first-person) in an asymmetrical multiplayer stealth game. Spies use stealth and gadgets while mercenaries rely on firepower to control the map in classic modes like bomb defusal.
